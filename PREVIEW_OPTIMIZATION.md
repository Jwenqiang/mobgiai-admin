# 预览对话框优化说明

## 优化内容

### 1. 全屏预览
- 将所有图片和视频预览对话框改为全屏显示（fullscreen）
- 移除了宽度限制和居中属性
- 对话框占据整个视口（100vw x 100vh）
- 背景色改为纯黑色（#000），提供更沉浸的预览体验

### 2. 禁止页面滚动
- 添加了 `watch` 监听器，监听预览对话框的打开/关闭状态
- 预览打开时：`document.body.style.overflow = 'hidden'`
- 预览关闭时：`document.body.style.overflow = ''`
- 防止在预览全屏时页面背景滚动

### 3. 关闭按钮优化
**位置和尺寸：**
- 位置：右上角（top: 32px, right: 32px）
- 尺寸：56px × 56px（原来是 48px）
- z-index: 10001（确保在最上层）

**样式优化：**
- 背景：半透明黑色 `rgba(0, 0, 0, 0.6)` + 毛玻璃效果
- 边框：2px 白色半透明边框
- 图标：24px 大小
- 悬停效果：放大 1.1 倍，背景变亮
- 点击效果：缩小到 0.95 倍
- 移除了旋转动画，改为简单的缩放效果

### 4. 左右箭头优化（仅 ImageGenerateView.vue）
**位置和尺寸：**
- 位置：垂直居中，左右各距离边缘 40px
- 尺寸：64px × 64px（原来是 56px）
- z-index: 10001

**样式优化：**
- 背景：半透明黑色 `rgba(0, 0, 0, 0.6)` + 毛玻璃效果
- 边框：2px 白色半透明边框
- 图标：28px 大小（原来是 24px）
- 悬停效果：放大 1.15 倍，背景变亮
- 点击效果：缩小到 0.95 倍

### 5. 图片计数器优化（仅 ImageGenerateView.vue）
**位置和尺寸：**
- 位置：顶部居中（top: 32px）
- 内边距：12px 28px（原来是 10px 24px）
- z-index: 10001

**样式优化：**
- 背景：半透明黑色 `rgba(0, 0, 0, 0.7)` + 毛玻璃效果
- 边框：2px 白色半透明边框
- 字体：16px，加粗，字间距 1px
- 字体家族：SF Pro Display（苹果风格）
- 颜色：纯白色 #ffffff

### 6. 媒体显示优化
- 移除了图片/视频的圆角效果（border-radius: 0）
- 移除了阴影效果
- 移除了悬停时的缩放效果
- 最大高度改为 100vh，充分利用全屏空间

## 设计理念

### 统一的视觉风格
- 所有控制按钮采用统一的半透明黑色背景
- 统一的毛玻璃效果（backdrop-filter: blur(20px)）
- 统一的白色半透明边框
- 统一的悬停和点击交互效果

### 简洁现代
- 移除了复杂的渐变和多重阴影
- 采用简单的纯色背景
- 清晰的层次关系
- 流畅的动画过渡

### 用户体验
- 更大的按钮尺寸，便于点击
- 清晰的视觉反馈
- 全屏沉浸式预览
- 防止意外滚动

## 浏览器兼容性

- 毛玻璃效果（backdrop-filter）在现代浏览器中支持良好
- 如需兼容旧版浏览器，可添加 fallback 样式
