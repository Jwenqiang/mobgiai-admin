# 图片上传功能演示指南

## 🎯 功能演示步骤

### 1. 启动应用
```bash
cd mobgiai-admin
npm run dev
```

### 2. 访问图片生成器
- 打开浏览器访问开发服务器地址
- 导航到图片生成器页面

### 3. 测试上传功能

#### 基础上传测试
1. 点击左上角的 "+" 按钮
2. 选择一张图片文件
3. 观察缩略图出现在文案输入框下方
4. 注意显示 "参考图片 (1/5)" 标签

#### 多图片上传测试
1. 重复上传操作，最多上传5张图片
2. 观察缩略图排列成网格布局
3. 当达到5张时，上传按钮变为禁用状态
4. 尝试再次上传，会显示限制提示

#### 删除功能测试
1. 将鼠标悬浮在任意缩略图上
2. 观察右上角出现红色删除按钮
3. 点击删除按钮移除图片
4. 观察计数器实时更新，上传按钮重新启用

#### 预览功能测试
1. 点击任意缩略图（避开删除按钮）
2. 观察弹出预览对话框显示原图
3. 点击对话框外部或关闭按钮退出预览

### 4. 测试悬浮面板

#### 生成图片后的悬浮面板
1. 输入文案并点击生成按钮
2. 等待生成完成后，观察底部悬浮面板
3. 在悬浮面板中测试上传、删除、预览功能
4. 注意悬浮面板中的缩略图更小（50x50px）

### 5. 响应式测试

#### 移动端适配
1. 调整浏览器窗口到移动端尺寸
2. 观察布局自动调整
3. 测试触摸操作的响应性

## 🐛 常见问题排查

### 上传不成功
- 检查文件格式是否为图片
- 检查文件大小是否超过10MB
- 检查是否已达到5张限制

### 删除按钮不显示
- 确保鼠标正确悬浮在缩略图上
- 检查CSS z-index是否正确

### 预览不工作
- 确保点击的是缩略图而非删除按钮
- 检查图片URL是否有效

### 响应式问题
- 测试不同屏幕尺寸
- 检查CSS媒体查询是否生效
