# 图片生成器底部下拉框更新

## 更新内容

根据提供的图片界面，对图片生成器底部的下拉框进行了以下更新：

### 1. 选择比例（尺寸选择）
- 按照图片显示的顺序重新排列了比例选项：
  - 21:9 (2560×1080)
  - 16:9 (1920×1080) 
  - 3:2 (1536×1024)
  - 4:3 (1440×1080)
  - 1:1 (1024×1024)
  - 3:4 (1080×1440)
  - 2:3 (1024×1536)
  - 9:16 (1080×1920)

### 2. 新增分辨率选择
- 添加了分辨率选择下拉框
- 选项包括：
  - 高清 2K
  - 超清 4K
- 使用 Monitor 图标

### 3. 新增图片张数选择
- 添加了图片张数选择下拉框
- 选项包括：1、2、3、4张
- 默认选择4张
- 使用 Grid 图标
- 采用网格布局显示选项

### 4. 界面优化
- 保持了原有的即梦风格设计
- 所有新增选择器都支持弹窗选择
- 添加了悬停动画效果
- 支持响应式布局

### 5. 功能增强
- 生成图片数量根据选择的张数动态调整
- 图片展示网格根据数量自动调整布局：
  - 1张：单列居中显示
  - 2张：2列网格
  - 3张：3列网格  
  - 4张：4列网格
- 移动端自动适配为2列布局

## 技术实现

### 新增接口定义
```typescript
interface Resolution {
  value: string
  label: string
  quality: string
}

interface ImageCount {
  value: number
  label: string
}
```

### 新增响应式数据
- `resolutions`: 分辨率选项数组
- `currentResolution`: 当前选择的分辨率
- `imageCounts`: 图片张数选项数组
- `currentImageCount`: 当前选择的图片张数

### 新增方法
- `selectResolution()`: 选择分辨率
- `selectImageCount()`: 选择图片张数

### 样式更新
- 添加了分辨率和图片张数选择器的样式
- 更新了图片网格的动态布局样式
- 保持了统一的设计风格

## 使用说明

1. 用户可以通过底部的参数选择区域选择：
   - 生成模型
   - 图片尺寸比例
   - 艺术风格
   - 分辨率质量
   - 生成图片张数

2. 所有选择都会实时更新界面显示

3. 生成的图片数量和展示布局会根据选择自动调整

## 兼容性

- 支持桌面端和移动端
- 保持了原有的所有功能
- 向后兼容现有代码
