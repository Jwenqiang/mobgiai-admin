# 视频生成页测试说明

## 功能概述

视频生成页现在支持两种显示模式：
1. **无内容模式**：显示居中的输入界面（类似图片生成页）
2. **有内容模式**：显示生成的视频列表和底部悬浮输入面板

## 测试控制

### 方法1：通过页面右上角的测试按钮
- 页面右上角有一个测试控制按钮
- 点击可以快速切换"无内容模式"和"有内容模式"
- 按钮会显示当前状态

### 方法2：通过代码修改
在 `VideoGenerateView.vue` 文件中找到：
```javascript
const hasContent = ref(false) // 改为 true 可以测试有内容的状态
```

- 设置为 `false`：显示无内容时的居中布局
- 设置为 `true`：显示有内容时的列表布局

## 布局特点

### 无内容模式（hasContent = false）
- 页面标题："视频生成" + 视频图标
- 居中的输入区域：
  - 上传按钮（简化版）
  - 提示词输入框
  - 参数选择按钮（模型、尺寸、画质、时长等）
  - 生成按钮
- 完全居中布局，上下左右居中

### 有内容模式（hasContent = true）
- 显示生成的视频卡片列表
- 底部悬浮输入面板
- 支持视频操作（重新编辑、再次生成、下载、保存等）

## 样式一致性

视频生成页的无内容布局与图片生成页保持一致：
- 相同的背景渐变
- 相同的居中布局逻辑
- 相同的输入框和按钮样式
- 相同的参数选择器样式

## 开发建议

1. 开发时可以通过右上角按钮快速切换状态
2. 测试无内容状态时，确保所有功能正常工作
3. 测试有内容状态时，验证视频列表和操作功能
4. 上线前记得将 `hasContent` 设置为合适的默认值
